사용 방법

1. 설문 데이터 입력

이름과 다지선다형 설문(복수 선택 가능)을 실시한 후, 왼쪽 테이블에 이름과 각 응답자가 선택한 설문 요소를 입력하세요. 설문 결과는 하나의 셀에 여러 요소를 입력하며, 각 요소는 쉼표(,) 로 구분합니다. 예: "영화, 음악, 독서"

2. 관계망 데이터 생성

이름 관계망: 각 사람의 설문 결과를 바탕으로 자카드 유사도를 계산하여 사람들 간의 유사도를 측정합니다.
요소 관계망: 설문 결과의 각 요소들이 함께 나타나는 빈도(공출현)를 바탕으로 요소들 간의 관계를 측정합니다.
생성된 데이터는 수정하거나 삭제하여 그래프를 조정할 수 있습니다.

3. 관계망 그래프 그리기

"관계망 그래프 그리기" 탭으로 이동하여 생성된 관계망 데이터를 시각화합니다.
집단 찾기: Louvain 알고리즘을 사용하여 네트워크의 커뮤니티를 감지합니다.
중심성 계산: 노드의 연결 중심성과 고유벡터 중심성을 계산합니다.
